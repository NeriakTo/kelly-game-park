# 🎮 Kelly's Game Park

為國小兒童設計的益智遊戲平台，動物森友會風格介面，柔和配色搭配圓角卡片設計。
所有遊戲皆為純前端實作，無需後端伺服器。

## 遊戲列表

| 遊戲 | 說明 | 核心機制 |
|------|------|---------|
| 🔢 **數獨** | 填入數字，訓練邏輯推理 | 回溯法生成 + 唯一解驗證，依難度挖洞 22–54 格 |
| ♟️ **中國象棋** | 與 AI 對弈，策略思考 | Minimax + Alpha-Beta 剪枝，3 級搜尋深度 |
| 🃏 **記憶翻牌** | 翻開配對，訓練記憶力 | 4–12 對 emoji 卡片，依難度調整數量 |
| ⌨️ **中/英打字練習** | 練習速度與正確率 | 鍵位課程（注音/英文）+ 短文模式，即時 CPM 計算 |
| 🧮 **數學挑戰** | 依年級出題，邊玩邊學 | 1–6 年級課綱對應，單元追蹤 + 錯題本 |
| 🦕 **達克比的恐龍商店** | 用硬幣買恐龍，練習金額計算 | 9 種題型（認識硬幣→找零→打折），硬幣互動付款面板，化石收藏獎勵 |
| ⚫ **圍棋教室** | 從零開始學圍棋 | 完整圍棋引擎（提子/打劫/數地），8 堂課程教學 + 3 級 AI 對弈 |

## 難度等級

全遊戲共用五段難度，對應學習階段：

| 等級 | 對象 | 代表 |
|------|------|------|
| ⭐ | 國小低年級（1–2 年級） | 基礎概念、小數字 |
| ⭐⭐ | 國小中年級（3–4 年級） | 進階運算、策略入門 |
| ⭐⭐⭐ | 國小高年級（5–6 年級） | 複合題型、深度思考 |
| ⭐⭐⭐⭐ | 國中 | 抽象化與多步驟推理 |
| ⭐⭐⭐⭐⭐ | 高中 | 進階邏輯與高難度題型 |

## 專案結構

```
src/
├── components/
│   ├── GameCard/          # 首頁遊戲卡片
│   ├── Layout/            # 響應式佈局（桌面側欄 / 手機底欄）
│   └── UI/                # 共用元件（DifficultySelector）
├── games/
│   ├── chess/             # 中國象棋（engine + UI）
│   ├── dino-shop/         # 恐龍商店（題型生成 + 硬幣互動 + 收藏系統）
│   ├── go/                # 圍棋（引擎 + 課程 + AI + 棋盤渲染）
│   ├── math/              # 數學挑戰（課綱題庫 + 錯題本）
│   ├── memory/            # 記憶翻牌
│   ├── sudoku/            # 數獨（生成器 + 唯一解驗證）
│   └── typing/            # 打字練習（鍵位課程 + 短文模式）
├── pages/                 # 各遊戲頁面入口
├── stores/                # Zustand 全域狀態（成績 / 難度 / 設定檔）
├── types/                 # 共用型別定義
└── utils/                 # 工具函式（shuffle 等）
```

## 設計原則

- **ADHD 友善**：短回合（3–5 分鐘）、即時正回饋、不懲罰錯誤
- **漸進式學習**：從簡單到複雜，每個遊戲都有引導機制
- **純 Emoji + CSS 繪製**：不依賴外部圖片資源
- **響應式設計**：桌面（側欄導覽）、平板（頂部導覽）、手機（可捲動底欄）
- **資料持久化**：成績、錯題本、課程進度、收藏皆存於 localStorage

## AI 題目模式（Phase 2）

- 設定頁可輸入 **OpenAI / Gemini API Key** 啟用 AI 題目生成。
- 數學挑戰與恐龍商店採「**本地先出題、AI 背景覆寫**」策略，避免等待卡住遊戲。
- 若 AI 回應逾時、格式錯誤或供應商異常，會自動回退到本地題庫，遊戲不中斷。
- 介面會顯示題目來源（AI / 本地題庫）。
