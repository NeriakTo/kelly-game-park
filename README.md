# 🎮 Kelly's Game Park

為兒童設計的益智遊戲平台，動物森友會風格介面。

## 遊戲列表

- 🔢 **數獨** — 邏輯推理訓練，自動生成 5 個難度
- ♟️ **中國象棋** — 與 AI 對弈，minimax 引擎
- 🃏 **記憶翻牌** — 圖案配對記憶訓練
- 🧮 **數學挑戰** — 即將推出
- 🎯 **2048** — 即將推出

## 技術棧

React 19 + TypeScript + Vite + Tailwind CSS v4 + Zustand + Framer Motion

## 開發

```bash
npm install --include=dev
npm run dev
```

## 部署

推送到 `main` 分支會自動部署至 GitHub Pages。
