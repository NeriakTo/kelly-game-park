# Kelly Game Park — Phase 2.4 E2E Smoke Checklist

日期：2026-03-08

## 測試目標
驗證「無 API Key / 有 API Key / API 失敗」三種情境下，數學挑戰與恐龍商店皆可完整遊玩。

---

## Case A：無 API Key（預設）

### 操作
1. 進入設定頁，確認 API Key 欄位為空。
2. 進入「數學挑戰」開始 3 題。
3. 進入「恐龍商店」開始 3 題。

### 預期
- 不會報錯或卡住。
- 出題來源顯示「本地題庫」。
- 答題流程、計分、下一題皆正常。

---

## Case B：有效 API Key

### 操作
1. 設定頁填入有效 OpenAI 或 Gemini Key。
2. 在數學挑戰連續玩 5 題。
3. 在恐龍商店連續玩 5 題。

### 預期
- 多數題目可看到來源為「AI」。
- 若個別題目回退，仍可無縫作答。
- 恐龍商店 `coin payment` 題型仍維持本地邏輯可玩。

---

## Case C：API 失敗（壞 Key / 斷網）

### 操作
1. 設定頁填入錯誤 Key，或斷開網路。
2. 分別進入數學挑戰與恐龍商店。

### 預期
- 可繼續遊玩、不白屏。
- 出題來源顯示「本地題庫（AI 暫不可用，自動回退）」。
- 無需重整頁面即可持續出題。

---

## 建置與部署驗證

```bash
npm run build
```

預期：build 成功，並可將 `dist/` 同步至 `docs/`（GitHub Pages）。
